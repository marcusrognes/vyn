Tutorials are step-by-step walkthroughs of building a complete app. Pick one and follow it; each tutorial is self-contained.

Available now:

- [Build a todo app](/vyn/tutorials/build-a-todo/) — a complete realtime todo app: model, actions, client, subscriptions, and MCP exposure
  in under 200 lines.
- [Build notes with auth](/vyn/tutorials/build-notes-with-auth/) — a multi-user notes app with SQLite persistence, session-cookie auth,
  per-user scoping, parameterized routes, and realtime updates.
- [Build a research notebook](/vyn/tutorials/build-a-research-notebook/) — multi-page tutorial: MongoDB persistence, SuperJSON for rich
  types, a streaming AI agent, scheduled deep-research jobs, multi-channel notifications, and a Tailwind UI.
