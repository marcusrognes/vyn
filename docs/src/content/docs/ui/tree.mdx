`@vyn/ui/tree` turns a nested `<ul>` into a fully-keyboarded tree view.
Every `<li>` becomes a `treeitem`; any `<li>` whose direct child is a
`<ul>` becomes an expandable folder. The behavior wires roles,
`aria-expanded`, roving tabindex, and the arrow-key conventions you
expect from a file tree.

```ts
import "@vyn/ui/tree";
```

## Keyboard

| Key | Effect |
|---|---|
| ↑ / ↓ | Move focus to the previous / next **visible** item |
| → | Expand the current folder, or move to its first child if already open |
| ← | Collapse the current folder, or move to its parent if already closed (or a leaf) |
| Home / End | Move to the first / last visible item |
| Enter, Space | Toggle a folder, or fire `activate` on a leaf |

## Attributes

Applied by the behavior — you don't usually write these yourself, but
they're observable and overridable:

| Element | Attribute | What it does |
|---|---|---|
| Root `<ul>` | `data-tree` | Activates the behavior |
| Root `<ul>` | `role="tree"` | Set automatically |
| Every `<li>` | `role="treeitem"` | Set automatically |
| Folder `<li>` (direct child `<ul>`) | `aria-expanded="true" \| "false"` | Set automatically; toggle to start expanded |
| Folder's child `<ul>` | `role="group"` | Set automatically |
| Leaf `<li>` | `data-value="..."` | Surfaced in the `activate` event detail |

## Events

| Event | Detail | When |
|---|---|---|
| `toggle`   | `{ expanded: boolean, item: HTMLElement }` | Bubbles from a folder when expanded/collapsed |
| `activate` | `{ item, value }` | A leaf is clicked or activated with Enter/Space |

## Composing

- The CSS rule that actually hides collapsed children lives in your
  stylesheet — the behavior only flips the `aria-expanded` attribute.
  This keeps the DOM stable (no items removed/inserted) so other
  behaviors layered on top (selection, sortable, etc.) keep their
  state.
- Combine with [`@vyn/ui/select`](/vyn/ui/select/) on the tree root
  for single-leaf-selection, or with [`@vyn/ui/scroll-into-view`](/vyn/ui/scroll-into-view/)
  to keep the focused item visible in a tall sidebar.
