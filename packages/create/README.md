# @vynjs/create

Scaffolder for [Vyn](https://github.com/marcusrognes/vyn) apps.

```sh
deno create jsr:@vynjs/create my-app
cd my-app
deno task dev
```

Equivalent invocations:

```sh
deno run -A jsr:@vynjs/create my-app
deno run -A jsr:@vynjs/cli init my-app   # what this package delegates to
```
